Detailed Description

template<unsigned int MAX, unsigned int MIN = 1>
class yijia::factorial::Factorial< MAX, MIN >::Iterator

Use the type Iterator in class Facotial to get factorial value that you want.

Offer a random access to index_ and factorial_.

  • A contanier of math factorial value
    • You can set the max and min index of factorial
  • Accessing to factorial value is supported by inline class Iterator.
